PXLME.js
========

#### Description ####

PXLME (Pixel Me) is an open source framework for dynamic pixel grafics by Tobias Schultka.

### Simple Usage ###

```javascript
	
	// create a simple stage without any arguments
	var stage = new PXLME.Stage({});
```
